About This Application

This Movie Catalog application is built with:

  • Frontend: Angular with Bootstrap
  • Backend: Spring Boot
  • Database: SQL database for storing movies and user data
  • External API: Integration with OMDB API for movie information

The application demonstrates a full-stack implementation with authentication, authorization, and CRUD operations.
